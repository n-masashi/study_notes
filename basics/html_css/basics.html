CSSの基礎

CSSはデザインを担当する。色、サイズ、レイアウトを指定します。
例： `color: red;` で文字色を赤にする

よく使うタグ

  • <header>：ヘッダー領域
  • <nav>：ナビゲーション
  • <main>：主なコンテンツ
  • <article>：独立した記事
  • <section>：セクション分け
  • <footer>：フッター
