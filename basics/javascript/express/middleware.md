# ミドルウェアの仕組みと next() の説明

## ミドルウェアとは？

- リクエストとレスポンスの間に挟む処理
- 例：ログ記録、認証チェック、リクエスト解析など

## app.use() の役割

- すべてのリクエストに対して処理を行う
- 例：

~~~js
app.use((req, res, next) => {
  console.log('アクセスがありました');
  next(); // 次の処理に進む
});
~~~

## next()とは？

- 次のミドルウェアやルート処理に渡すための関数
- これを呼ばないと処理が止まる

## ミドルウェアの種類

- 組み込みミドルウェア（express.json()など）
- 外部ミドルウェア（cookie-parserなど）
- 自作ミドルウェア
