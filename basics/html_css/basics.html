HTMLの基礎

HTMLは「構造」を作るための言語。文章の見出し・段落・リンクなどを記述します。
例：<h1> は大見出し、 <p> は段落、 <a> はリンク

CSSの基礎

CSSはデザインを担当する。色、サイズ、レイアウトを指定します。
例： color: red; で文字色を赤にする

よく使うセマンティックタグ

  • <header>：ヘッダー領域（タイトルやロゴなど）
  • <nav>：ナビゲーションメニュー
  • <main>：主要コンテンツ領域（一つだけ）
  • <article>：独立した記事や投稿のまとまり
  • <section>：テーマごとの区切り（セクション）
  • <footer>：フッター領域（著作権など）
  • <figure>：画像や図のまとまり（説明文とセット）
  • <figcaption>：figureのキャプション（説明文）
  • <aside>：補助的な内容（サイドバーや関連記事など）
  • <div>：意味を持たない汎用コンテナ（グループ化やレイアウト用）
  • <mark>：強調表示（ハイライト）
  • <time>：日時や時間の情報を表すタグ
  • <form>：フォームを表すタグ
